3 Class Options

In addition to any options passed to the memoir class (e.g. oldfontcommands; see Section 5.4, font size), the msu-thesis class takes six options: one to specify the type of degree, (see Table 1), one to enable landscape page numbering, two to enable per-chapter bibliographies and appendices, two to make small adjustments to Table of Contents. If no degree option is specified, a Ph.D. dissertation is assumed.
