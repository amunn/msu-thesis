3.1 The [lscape] option

The Formatting Guide requires that any lansdscape pages be numbered in landscape mode (i.e. along the long edge of the page) rather than in portrait mode, and rotated in the final PDF. The [lscape] option enables this. The option loads both the pdflscape package and tikz, and is implemented as a class option so that you are not required to load those packages if they are not needed.
